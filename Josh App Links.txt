# Full App & Game Review — Parris Portfolio
## Ranked: Closest to AAA Indie → Prototype/Stub

Scoring dimensions: Polish, Features, Functionality, Fun/Engagement, Completeness, AAA Potential

---

## TIER S — AAA Indie Candidate
*Could ship on itch.io or Product Hunt today and hold its own*

---

### 1. WHISPERING WILDS v2 (PyScript)
**URL:** https://joshuaparrisdadlan-stack.github.io/whispering-wilds/
**Score: 9.2 / 10**

The standout of the entire portfolio. Feature density is exceptional for a solo browser game.

**Features:**
- Full text adventure engine running Python in-browser via PyScript
- 5 playstyle profiles (Storybook → Hardcore)
- 5 guidance levels (Off → Hand-in-Hand)
- Granular difficulty tuning: encounter rate, flee success, XP gain, drop luck, vendor prices, puzzle hints — each individually adjustable
- Safety options (no random attacks, can't be attacked, hide attack verbs)
- Read Aloud + Simplify Text accessibility features
- Multiple themes (Dark Forest, Light Meadow)
- Dyslexic font toggle
- Compact mode
- Sound + Music toggles
- Leaderboards
- Vendor/shop system
- Quest tracking panel
- Inventory panel
- Map panel with display-on-move
- XP progress bar
- Export/Import save
- Random seed system
- Dev mode
- Parental controls with PIN lock
- Tutorial bar
- Arrow key navigation

**What holds it back from 10:** No visual art/assets, no audio currently active, no story cinematics or title screen. The PyScript load time is a friction point.

**AAA path:** Add a title screen, ambient audio, a hand-drawn map image, and a few key art pieces. This is already a better-designed game than most itch.io paid titles.

---

### 2. PARRIS TECH SERVICES APP (Windows)
**URL:** https://parris-tech-services-app.vercel.app/
**Score: 9.0 / 10**

The most functionally complete app in the portfolio. A legitimate business tool.

**Features:**
- Full client management (add, edit, search, filter, find duplicates, CSV import/export)
- Full appointment system (recurring, status tracking, device checklist, follow-up reminders)
- Invoice creation with line items, time blocks, tax (GST), presets, payment URL, Zoho Books integration
- Reconciliation view with aging report, bulk mark paid, finance summary, CSV export
- Templates & assets with variable substitution (`{{name}}`, `{{datetime}}`)
- Quick actions with deep links (Calendly, JoshHub, ParrisTechApp)
- ICS calendar export
- Analytics dashboard
- Encrypted backup export/import with passphrase
- Dry-run import for safety
- Onboarding tips system
- Themed branding (Default, Irish, Aussie, Halloween, Christmas)
- Logo upload
- Keyboard shortcuts (Ctrl+1-4, Ctrl+S)
- Help & services section with FAQs, changelog, cleanup checklists
- Windows + Mac cleanup checklists
- Local storage persistence
- Print invoice view
- Demo data seeding (3 modes)

**What holds it back from 10:** localStorage only (no cloud sync), no mobile-optimised layout, no actual payment processing.

---

### 3. PARRIS TECH APP (Mac)
**URL:** https://parris-tech-app.vercel.app/
**Score: 8.5 / 10**

A polished guided tech support session tool — purpose-built and well-executed.

**Features:**
- Session timer with remaining time
- Client profile creation with devices, past issues, contact info
- Session length + location (onsite/remote) settings
- Timezone support
- Session templates (save, delete, apply)
- Dynamic checklist with undo last, collapse completed
- Progress tracking (N of N done)
- Invoice line items with subtotal, tax, total
- End-of-visit summary generation
- Print view + PDF invoice export
- JSON export (all sessions)
- Import sessions
- Rules editor — build custom modules with steps, questions, risk notes, timebox
- Client profiles manager
- Follow-ups & reminders with ICS export and email draft generation
- Session history with client filter
- Reports with time range filter + CSV export
- Metrics dashboard
- Themes (Dark, Light, Green, Blue)
- Offline detection

**What holds it back:** Overlaps significantly with ParrisTechServicesApp — feels like it should be merged or clearly differentiated. No cloud sync.

---

### 4. LIFEHUB DASHBOARD
**URL:** https://lifehubdashboard.vercel.app/
**Score: 8.3 / 10**

Staggeringly feature-rich. The most ambitious single-page app in the portfolio.

**Features:**
- Multi-panel dashboard with 20+ named sections
- Kiosk mode with custom rotation, panel dwell time, drag-to-reorder, presets
- Focus mode + Calm mode
- Multiple themes (Modern, Classic, High Contrast, Irish, Aussie, Halloween, Christmas, Easter)
- Compact spacing, readable font, no shadows, pixel mode
- Copilot natural language search (`/` shortcut)
- Wellbeing snapshot (Welltory CSV upload, stress/energy/focus/HRV)
- Calendar panel with .ics upload + agenda
- Activity timeline (events, reminders, files, downloads, automations, wellbeing, backups)
- Inbox triage board (drag-to-column)
- Downloads watcher + auto-triage heuristics
- Automation scheduler with queue, dry run, presets, JSON import/export
- System health freshness checks
- Resurface carousel (90-day untouched files)
- Achievement system with badges
- Weekly LifeHub routine checklist
- Resource spotlight
- Playable projects launcher
- Project scratchpad with promote-to-LifeHub, copy, download .md
- Backup integrity panel
- D&D tools: skill checks, saves, ability scores, magic item attunement
- Character creator (race, class, background, XP)
- Mini adventure (Clocktower Heist — choice-based)
- Encounter tracker (inbox dragon, downloads goblins — gamified tasks)
- Dice roller (d4–d20, encounter, loot)
- Tavern banter rotator
- Daily quest with reroll + streak
- Text game (PyScript Wilds engine, 3 save slots, autosave)
- Text game v2 (expanded sandbox build)
- Quick actions with custom add + drag/pin
- Daily realm event
- File area deep-links
- Keyboard shortcuts (I, W, F, P, H, A, ?, /)
- Visitor of the Day (fun randomiser)

**What holds it back:** It's almost too much — navigation friction is high, many panels are empty on first load, local-only with no sync. The gamification and work tools compete for attention rather than complement.

### 7. NULL
**URL:** https://joshuaparrisdadlan-stack.github.io/Null/
**Score: 7.5 / 10**

The most conceptually distinctive game in the portfolio. Genuinely unusual design.

**Features:**
- Text adventure with command parser
- Rope/bead sensory mechanic ("Pings" system) — unique
- Bell/sensate tracker
- Known Routes map
- Inventory
- Logbook (event recording)
- Turn counter
- Save / Load / Restart
- Tactile design philosophy baked into UI (SVG bead assets)
- Accessibility-forward tone and content warnings

**What holds it back:** Content depth unknown without playing through. UI is minimal — intentionally so, but risks feeling sparse to a new player. Could use a short onboarding passage.

**AAA path:** The Pings mechanic is genuinely publishable as a concept. Expand the world, add ambient audio, and this is an art game that would get coverage.
